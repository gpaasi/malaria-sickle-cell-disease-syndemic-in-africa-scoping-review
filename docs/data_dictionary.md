# Data dictionary

## included_sources_characteristics.csv

- study_id: unique source identifier
- source: author, year, and short title
- country_or_region: source country or region as abstracted
- evidence_type_or_design: evidence type or study design
- population_data_source_or_focus: population or data source
- main_concept_addressed: main concept addressed
- domain_codes: D1 to D7 domain coding
- relevance_to_review_question: source relevance statement
- primary_geographic_category: derived geography for Figure 3
- primary_population_system_level: derived population/system level for Figure S1
- source_category: source type category for the main matrix
