# Malaria-SCD Syndemic Review Workflow

This repository contains the reproducible workflow for:

**Reframing the Malaria-Sickle Cell Disease Relationship in Sub-Saharan Africa as a Bounded Syndemic: A Scoping Review and Evidence Gap Map**

## Protocol registration

The review protocol was registered on the Open Science Framework:

https://doi.org/10.17605/OSF.IO/6TVSB

Associated OSF project: https://osf.io/8kdp5

## Contents

- `protocol/`: OSF registration and protocol summary
- `search_strategy/`: database, grey literature, and citation search documentation
- `screening/`: PRISMA counts and screening tracking files
- `extraction/`: source characteristics and coding files
- `analysis/`: scripts for generating summary tables and evidence maps
- `outputs/`: generated tables, figures, and supplementary files
- `manuscript/`: manuscript components
- `docs/`: data dictionary, coding rules, and reproducibility notes

## Reproducibility note

This repository does not include copyrighted full-text articles. It includes extracted data, coding decisions, and scripts needed to reproduce the review tables and evidence gap maps.
